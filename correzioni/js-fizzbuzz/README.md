# FIZZBUZZ

- PER OGNI NUMUERO da 1 a 100 controllare
  - SE numero è multiplo di 3 e di 5
    - stampare FIZZBUZZ
  - ALTRIMENTI SE numero è multiplo di 3
    - stampare FIZZ
  - ALTRIMENTI SE numero è multiplo di 5
    - stampare BUZZ
  - ALTRIMENTI
    - stampare il numero
